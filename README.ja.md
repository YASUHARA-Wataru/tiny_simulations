# tiny_simulations

ブログ記事や実験のために書いた、小さなシミュレーションコード集です。

## 目的

世界の一部を、もう少しはっきり理解するために、  
シンプルなシミュレーションを使っています。

重視している点は次の通りです。

- シンプルなモデル
- 仮定を明示すること
- 手早く試せること
- 1記事につき1コード

## 内容
```text
tiny_simulations/
├─ discussion/ 
├─ ec_sim/
├─ life_rhythm/
├─ monetary_policy/
├─ statistics/
├─ train/ 
├─ X_vs_threads/
└─ README.md
```
